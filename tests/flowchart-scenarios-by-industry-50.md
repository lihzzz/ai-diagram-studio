# 流程图生成场景库（按行业分类 50 条）

## 电商（10）

1. 生成电商下单流程：用户浏览商品 -> 加入购物车 -> 提交订单 -> 支付 -> 仓库出库 -> 物流配送 -> 确认收货
2. 生成电商退款流程：用户申请退款 -> 客服审核 -> 仓库验货 -> 财务退款 -> 消息通知 -> 工单关闭
3. 生成电商换货流程：用户提交换货申请 -> 客服确认 -> 逆向物流取件 -> 质检 -> 补发新商品 -> 用户签收
4. 生成促销活动发布流程：运营提报活动 -> 审批预算 -> 配置优惠规则 -> 预发布校验 -> 活动上线 -> 效果复盘
5. 生成秒杀活动流程：商品报名 -> 库存锁定 -> 流量预热 -> 准点开抢 -> 支付倒计时 -> 超时回补库存
6. 生成会员积分流程：用户消费 -> 积分发放 -> 积分查询 -> 积分兑换 -> 库存扣减 -> 兑换通知
7. 生成商家入驻流程：提交资料 -> 资质审核 -> 合同签署 -> 店铺开通 -> 商品上架 -> 运营培训
8. 生成库存补货流程：库存预警 -> 采购申请 -> 供应商确认 -> 到货入库 -> 质检上架 -> 库存同步
9. 生成跨境订单流程：下单支付 -> 海关申报 -> 国际干线运输 -> 清关 -> 国内派送 -> 妥投签收
10. 生成电商客诉闭环流程：用户投诉 -> 工单分派 -> 问题定位 -> 处理方案 -> 用户确认 -> 复盘改进

## 金融（10）

1. 生成银行开户流程：客户提交资料 -> 身份核验(KYC) -> 风险评级 -> 开户审批 -> 账户激活 -> 欢迎通知
2. 生成个人信贷审批流程：客户申请 -> 征信查询 -> 反欺诈校验 -> 审批评分 -> 授信决策 -> 放款
3. 生成信用卡申请流程：在线申请 -> 资料补全 -> 风险审核 -> 额度核定 -> 制卡 -> 寄送激活
4. 生成支付清算流程：交易发起 -> 支付鉴权 -> 渠道路由 -> 清算对账 -> 结算入账 -> 通知商户
5. 生成异常交易风控流程：交易监测 -> 规则命中 -> 人工复核 -> 临时冻结 -> 客户核实 -> 解冻或拒绝
6. 生成反洗钱监测流程：交易采集 -> 可疑模型识别 -> 合规审查 -> 上报监管 -> 案件归档
7. 生成理财购买赎回流程：产品浏览 -> 风险测评 -> 购买确认 -> 份额确认 -> 发起赎回 -> 资金到账
8. 生成保险理赔流程：客户报案 -> 材料提交 -> 理赔审核 -> 损失核定 -> 打款赔付 -> 案件关闭
9. 生成催收流程：逾期识别 -> 分层分案 -> 外呼提醒 -> 协商还款 -> 还款跟踪 -> 结清归档
10. 生成日终对账流程：交易汇总 -> 渠道对账 -> 差异定位 -> 人工复核 -> 调账处理 -> 对账完成

## 医疗（10）

1. 生成门诊就诊流程：线上预约 -> 到院签到 -> 分诊叫号 -> 医生问诊 -> 检查检验 -> 开药缴费 -> 取药离院
2. 生成急诊分诊流程：患者到达 -> 分级分诊 -> 紧急救治 -> 辅助检查 -> 医嘱处理 -> 留观或入院
3. 生成住院流程：门诊建议住院 -> 床位分配 -> 入院评估 -> 治疗执行 -> 日常查房 -> 出院结算
4. 生成手术流程：术前评估 -> 术前讨论 -> 麻醉准备 -> 手术执行 -> 术后恢复 -> 术后随访
5. 生成检验检查流程：医生开单 -> 采样检查 -> 实验室处理 -> 报告出具 -> 医生解读 -> 患者查看
6. 生成慢病管理流程：建档 -> 定期随访 -> 指标监测 -> 用药调整 -> 健康宣教 -> 复诊计划
7. 生成远程会诊流程：基层发起会诊 -> 病历上传 -> 专家排班 -> 远程会诊 -> 方案回传 -> 落地执行
8. 生成药品管理流程：采购入库 -> 批次登记 -> 科室领用 -> 用药记录 -> 库存盘点 -> 近效期预警
9. 生成院感事件上报流程：异常发现 -> 事件登记 -> 感控评估 -> 应急处置 -> 追踪监测 -> 结案复盘
10. 生成体检流程：预约体检 -> 到场登记 -> 项目检查 -> 异常复检 -> 总检报告 -> 健康建议

## 教育（10）

1. 生成招生报名流程：线索收集 -> 咨询跟进 -> 提交报名 -> 材料审核 -> 录取通知 -> 缴费入学
2. 生成新生入学流程：学籍注册 -> 分班分宿 -> 账号开通 -> 入学教育 -> 正式上课
3. 生成课程排课流程：教学计划制定 -> 教师匹配 -> 教室分配 -> 冲突检查 -> 发布课表 -> 调课管理
4. 生成作业管理流程：教师布置作业 -> 学生提交 -> 系统查重 -> 教师批改 -> 成绩反馈
5. 生成考试组织流程：命题 -> 审题 -> 排考场 -> 考试实施 -> 阅卷评分 -> 成绩发布
6. 生成学业预警流程：成绩采集 -> 预警规则触发 -> 导师干预 -> 家校沟通 -> 学习改进计划 -> 复评
7. 生成在线课程发布流程：课程策划 -> 内容制作 -> 试播审核 -> 正式发布 -> 学习跟踪 -> 课程迭代
8. 生成教师教研流程：提出课题 -> 立项评审 -> 教学实验 -> 结果分析 -> 教研分享 -> 成果沉淀
9. 生成学费缴费流程：账单生成 -> 通知学生 -> 在线支付 -> 对账确认 -> 开票归档
10. 生成毕业审核流程：学分核验 -> 实习验收 -> 论文答辩 -> 综合评定 -> 审批通过 -> 发放证书

## 制造（10）

1. 生成制造订单交付流程：接单评审 -> 产能评估 -> 生产计划 -> 生产执行 -> 质检 -> 包装发货 -> 客户签收
2. 生成生产排程流程：需求预测 -> MPS主计划 -> 物料齐套校验 -> 工单下发 -> 进度跟踪 -> 异常重排
3. 生成设备点检流程：点检计划 -> 现场巡检 -> 异常登记 -> 维修派工 -> 复检确认 -> 记录归档
4. 生成质量异常处理流程：发现不良 -> 隔离批次 -> 原因分析 -> 纠正措施 -> 复验 -> 关闭8D报告
5. 生成物料领用流程：工单触发 -> 仓库备料 -> 领料出库 -> 产线签收 -> 用量回传 -> 库存更新
6. 生成新品导入NPI流程：需求评审 -> 样机开发 -> 小批试产 -> 工艺优化 -> 量产评审 -> 正式量产
7. 生成工程变更ECN流程：变更申请 -> 技术评估 -> 成本评估 -> 审批发布 -> 切换实施 -> 效果验证
8. 生成安全事故处理流程：事故上报 -> 应急响应 -> 现场处置 -> 原因调查 -> 整改闭环 -> 安全培训
9. 生成采购到付款流程：采购申请 -> 询价比价 -> 下采购单 -> 到货验收 -> 发票校验 -> 财务付款
10. 生成售后返修流程：客户报修 -> 故障诊断 -> 回厂返修 -> 复测验收 -> 返还客户 -> 满意度回访
